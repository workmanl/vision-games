## ğŸ“ Vision Exercise Principles

These games incorporate several key vision exercise techniques:

- **Visual Tracking**: Following moving objects (Space Explorer, Balloon Adventure)
- **Visual Memory**: Remembering and matching patterns (Memory Master)
- **Saccadic Eye Movements**: Quick eye movements between targets (All games)
- **Hand-Eye Coordination**: Timing clicks/taps with visual targets (All games)
- **Spatial Awareness**: Navigating spaces and avoiding obstacles (Nitro Snail Runner, Treasure Maze)
